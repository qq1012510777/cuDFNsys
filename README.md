# cuDFNsys

The _cuDFNsys_ is an open-source C++ library for DFN simulation, based on CUDA. It is also able to simulate flow in DFN.

# Prerequisites

The _cuDFNsys_ relies on several packages: Eigen, Gmsh, Matlab, UMFPACK.
